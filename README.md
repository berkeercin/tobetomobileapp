# TOBETO Clone Projesi

- [Berke Erçin](https://github.com/berkeercin)
- [Tuğçe Kahya](https://www.github.com/kahyatugce)
  kişileri tarafından geliştirilmiştir.

![tobeto-logo2](https://github.com/berkeercin/tobetomobileapp/assets/90865303/b848966f-a030-4b0c-80f3-bbac27d52da0)

Tobeto, "headhunting" yaklaşımını "headfarming" olarak değiştirmeyi hedefleyen, eğitim ve gelişim odaklı dijital bir platformdur. İstanbul Kodluyor projesi, Mobil Uygulama Geliştirici(Flutter) eğitimi kapsamında bitirme projesi, Pair 4 ekibi olarak Tobeto Eğitim Platformu'nun mobil uygulamasını başarıyla tamamlamıştır.
